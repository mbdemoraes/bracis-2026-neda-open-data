# 2026_bracis_neda
Anonymous repository for the submission "Towards a Noisy Estimation of Distribution Algorithm for Expensive Binary Optimization Problems". It contains:

1. **source_code.zip**: the Python source code used in the experimental setup.
2. **results.zip**: the experimental results in `.csv` format.
3. **opt_uk.zip**: the global optima for the UK test instances in `.csv` format.
4. **best_params.zip**: the best hyper-parameter combinations found using grid search, in `.csv` format.
